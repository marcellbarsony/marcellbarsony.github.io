Server-side Request Forgery [SSRF]

2024-06-17
:: webapp :: portswigger :: burp-suite :: server-side :: ssrf ::

Server-side request forgery (SSRF) allows an attacker to cause the server-side application to make a HTTP requests to an unintended, arbitrary internal or external) location.
